# Getting started

Essential information to help you get set up with Eternity.

## Beginner Guide
EOG has a [Beginner Guide](https://www.eog.gg/games/sword-x-staff/guides/beginner-guide/) for the first few days that's decent. In essence, try to save most resources after your class up, and don't expect to switch class before Aqualis (~level 100 I think?).

Key Notes:
* Save up **300 Destiny Fruit** for an event later
* Just return home if defeated outside a dungeon, food is a waste.
* Rolla is a top choice for burning stamina
* Gacha is limited by class - Higher class = better reward pool

## Codes
Check [EOG Historic Codes](https://www.eog.gg/games/sword-x-staff/#codes), then check the discord for new ones!
